SQL query cheatsheet in one page

To Create a database

  • CREATE DATABASE databasename;

To Delete a database

  • DROP DATABASE databasename;

To Work in a database

  • USE databasename;

To Create table in a database

  • CREATE TABLE table_name (
    column1 datatype,
    column2 datatype(size),
    column3 datatype(size)
);

To Delete a table

  • DROP TABLE tablename;

To show schema(structure) of a table

  • DESC tablename;

To Change name of a table

  • ALTER TABLE tablename 
RENAME TO new_table_name;

To Change name of a column

  • ALTER TABLE table_name
RENAME COLUMN old_name TO new_name;

To add new column at last

  • ALTER TABLE table_name
ADD column_name datatype(size);

To add new column at specific position

  • ALTER TABLE tablename 
ADD new_column_name datatype(size) 
AFTER column_name;

To Delete a column

  • ALTER TABLE table_name
DROP COLUMN column_name;

To modify datatype(size)/position of a column

  • ALTER TABLE tablename 
MODIFY COLUMN column_name datatype(size) 
AFTER column_name;

To insert values in a table

    To insert in a custom way 
  • INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);
    • To insert in a squence 
  • INSERT INTO table_name
VALUES (value1, value2, value3, ...);

To update values in a table

  • UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

To read values in a table

  • SELECT column1, column2, ...
FROM table_name
WHERE condition;
  • SELECT * FROM table_name
WHERE Country='Mexico';

Conditional Operators for where clause

    Operator Description
    = Equal
    > Greater than
    < Less than
    >= Greater than or equal
    <= Less than or equal
    <> Not equal. Note: In some versions of SQL this operator may be written as !=
    BETWEEN Between a certain range
    LIKE Search for a pattern
    IN To specify multiple possible values for a column

And OR and NOT Operators

  • SELECT column1, column2, ...
FROM table_name
WHERE condition1 AND condition2 AND condition3 ...;
  • SELECT column1, column2, ...
FROM table_name
WHERE condition1 OR condition2 OR condition3 ...;
  • SELECT column1, column2, ...
FROM table_name
WHERE NOT condition;

MIN() MAX() AVG() COUNT() and SUM()

  • SELECT MIN(column_name)
FROM table_name
WHERE condition; 
  • SELECT MAX(column_name)
FROM table_name
WHERE condition; 
  • SELECT COUNT(column_name)
FROM table_name
WHERE condition;
  • SELECT AVG(column_name)
FROM table_name
WHERE condition;
  • SELECT SUM(column_name)
FROM table_name
WHERE condition;

Order by (Ascending or Descending order)

  • SELECT * FROM Customers
ORDER BY Country DESC;
  • SELECT * FROM Customers
ORDER BY Country;

IN and BETWEEN

  • SELECT column_name(s)
FROM table_name
WHERE column_name IN (value1, value2, ...);
  • SELECT column_name(s)
FROM table_name
WHERE column_name IN (SELECT STATEMENT);
  • SELECT column_name(s)
FROM table_name
WHERE column_name NOT IN (value1, value2, ...);
  • SELECT column_name(s)
FROM table_name
WHERE column_name BETWEEN value1 AND value2;
